



## 📗 Discord TTS 봇
> 디스코드 TTS 봇

---

#### 📖 개요
디스코드에서 입력한 메시지를 음성으로 변환하여 읽어주는 봇입니다.
개인적으로 사용하기 위해 개발한 봇으로 실제 사용을 위해서는 네이버 클라우드 플랫폼의 Clova Voice 서비스를 신청하여 API를 발급 받아야 합니다.

#### 📄 명령어
```
* /니버보이스 - TTS 봇을 음성 채널에 초대합니다.
* /니버보이스종료 -TTS 봇을 음성 채널에서 내보냅니다.
```
